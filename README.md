> E-commerce de hardware con +20 módulos en producción (proyecto personal).

- 🔌 **~75 endpoints REST** · **30k+ LOC backend** · **~45 tests automatizados**
- 🔐 Auth JWT + Google OAuth con RBAC (CLIENT / ADMIN / SUPER_ADMIN) y guardas personalizadas
- ⚙️ Jobs asíncronos con Bull/Redis: emails, cron jobs con distributed lock para carritos abandonados
- 🤖 Automatización de workflows con N8N · Panel admin con métricas en Recharts
- 🛠️ NestJS · Next.js 15 · React 19 · PostgreSQL · Redis · Bull · MercadoPago

🔗 [Demo](https://world-assembly-technology.vercel.app) · [Backend](https://github.com/LucioMoran-dev/Back-End-Ecommerce-Tech) · [Frontend](https://github.com/LucioMoran-dev/WorldAssemblyTechnology)
